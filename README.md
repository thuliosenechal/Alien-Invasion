# Alien-Invasion
Alien invasion: Python crash course


## Environment (Tested):

OS: Win 10

Python: Python3.8

## Requirements:

Install Pygame module


## Game commands:

- Click play to start the game
- Space bar to shoot
- left and right arrow button to move the ship
